CSS像素(px)转vw单位工具

此工具可将CSS中的px单位转换为vw单位，使您的CSS更具响应性。输入您的CSS代码和设计稿宽度，工具将自动进行计算并输出转换后的结果。

转换规则：仅转换包含px单位的属性，其他属性将被忽略。计算公式：vw值 = px值 / 设计稿宽度 * 100。

工具使用专业的CSS解析库，能处理复杂的CSS语法和选择器。

示例：

输入CSS代码（设计稿宽度：1280px）：

.youtube-video-slider-container .swiper-scrollbar {
  height: 6px;
  background-color: rgba(212, 141, 2, 0.1);
  border-radius: 3px;
  position: relative;
  margin-top: 15px;
}
